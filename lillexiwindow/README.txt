1. This text editor can be used to insert chars, rects and images.
2. This text editor can insert and delete glyphs at a cursor, as well as images and rectangles. You can compose.
3. This text editor activates a horizontal and vertical scroll when necessary. (Just press enter until you're out of the 
screen and it shows up).
4. This text editor can undo and redo actions on the text panel.
5. You can change the font, the text color, and the text size.
